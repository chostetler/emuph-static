Answer: CAPTAIN JACK SPARROW

Written by Hannah Leaman

All that needs to be done in this puzzle is to identify the EMU building in the image. As implied by the flavor text, this uses the names of the buildings from the EMU Interactive Map.

Indexing into the names of the buildings, we get DEPP'S FAMOUS MOVIE ROLE. The thematic answer for this is CAPTAIN JACK SPARROW.
